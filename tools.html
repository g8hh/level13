What is this page?

This page contains some tools and cheats for players who have been affected by bugs and balancing issues in the game. The cheats here are quite specific, so unless you've been directed here to fix a specific issue, they will likely not be useful to you.

Load save

Copy paste the save you want to fix to the "Input" box. You can export your save from the game via the import/export popup in the footer. Once a fix has been applied, the fixed save will appear in the "Output" box. You can import it to the game using the same popup.

PROBLEM: Stuck on level 12 with not enough evidence

If you played version 0.3.1 and bought all other available upgrades before Compass, you might have gotten stuck without enough evidence to buy Compass. This tool will remove the Knife upgrade from you and reinburse the amount of Evidence you spent on it, allowing you to continue.
